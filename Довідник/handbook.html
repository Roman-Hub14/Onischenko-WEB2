MІЙ ЕЛЕКТОРННИЙ ДОВІДНИК HTML
Призначення Назва тегу Короткий опис Стандарт
Для каркасу документу <html> Початок і закінчення HTML-документа HTML5
<head> Eлемент-контейнер для метаданнuх HTML-документа
<body> Зберігання вмісту Веб-сторінки, відображається у вікні браузера
Для роботи з текстом <p> Параграфu y текстi
<em> З наклоном, звертає увагу
<i> Виділяє уривок тексту курсивом, не призначаючи йому додатковий акцен
<u> Виділяє уривок тексту підкресленням, без додаткового акценту
<h1>-<h6> Створює заголовки різних розмірів
<br> Перенесення тексту на новий рядок
<del> Позначає текст як віддалений, перекреслюючи його
<ul> Неупорядкований список
<ol> Упорядкований список
Для гиперссылок <a> Створює гіпертекстові посилання
